# Figma electron app (unofficial)

This package, its a web wrapper of the [Figma-web-app](https://figma.com) packed usig [Electron](http://electron.atom.io). It lets you install Figma on every distro who accept Snap and easily access it from system men√∫s or the Ubuntu Dash, instead of your web browser.

## install
```
sudo dpkg -i figma-linux_0.0.2_amd64.deb
```

## Run
```
git clone git@github.com:ChugunovRoman/figma-linux.git
cd figma-linux
npm i && npm start
```

## [Download](https://github.com/ChugunovRoman/figma-linux/releases)

PS: After install and authorized in Figma app, you need press the F5 button.
